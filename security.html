Security is not a feature.

Security architecture

🔒

Data at Rest

AES-256 encrypted volumes + PostgreSQL encryption. All database contents, audit logs, and user data protected.

🔐

Data in Transit

TLS 1.3 for all connections. Network communication encrypted end-to-end.

🗝️

API Key Storage

Fernet symmetric encryption for customer API keys. Never stored in plain text.

👤

Password Storage

Bcrypt hashing with salt. User credentials are irreversible — even we cannot read them.

🏢

Authentication

Built on Keycloak — used by NASA and Red Hat. Enterprise identity management with MFA support.

🛡️

DLP Engine

Microsoft Presidio — open-source, enterprise-grade PII detection. 50+ data types detected automatically.

Our data privacy commitment

  • We never store your conversations on our servers
  • We never see your employees' prompts
  • We never access your audit logs without explicit permission
  • We never sell, share, or analyze your company's data
  • We cannot access your installation remotely without your invitation
  • Your data is yours. Always.
